Notice d'utilisation

Presentation

Ce generateur de sudokus pedagogiques fonctionne entierement hors ligne. Il suffit d'ouvrir le fichier index.html dans un navigateur.

Comment utiliser

  • Taille : 4x4 pour les plus jeunes, 6x6 pour CP/CE1, 9x9 pour le primaire.
  • Niveau : "Tres facile" = beaucoup d'indices, ideal pour debuter.
  • Theme : chiffres, lettres, formes, animaux, fruits, couleurs, etoiles, vehicules, ou vos images.
  • Fiches : generez 1 a 6 fiches differentes d'un coup.

Boutons

  • Generer / Nouvelle fiche : cree de nouvelles grilles.
  • Correction : affiche/masque les cases manquantes en bleu.
  • Imprimer : lance l'impression. Les boutons sont masques automatiquement.
  • Plein ecran : ideal pour le tableau numerique.

Images importees

Selectionnez "Images importees", puis chargez autant d'images que la taille de la grille.

Impression

Plusieurs grilles tiennent sur une page A4 (surtout en 4x4). Les bordures s'impriment sans avoir besoin de cocher "Imprimer l'arriere-plan".

Conseils pedagogiques

  • Commencez par le 4x4 tres facile avec des formes ou des animaux.
  • Laissez les enfants manipuler des jetons avant de passer au papier.
  • Imprimez le corrige a part pour l'autocorrection.
